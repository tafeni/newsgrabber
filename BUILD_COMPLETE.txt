METHOD 1: Automated Setup
--------------------------
Windows:
   cd C:\xampp\htdocs\newsgrabber
   setup.bat

Linux/Mac:
   cd /path/to/newsgrabber
   chmod +x setup.sh
   ./setup.sh


METHOD 2: Manual Setup (5 Commands)
------------------------------------
1. composer install && npm install
2. cp .env.example .env && php artisan key:generate
3. # Edit .env with your database credentials
4. php artisan migrate && php artisan db:seed
5. npm run build && php artisan serve

-----------------------

================================================================================
RUNNING THE APPLICATION
================================================================================

Open 3 terminals and run:

Terminal 1:  php artisan serve             (Web server)
Terminal 2:  php artisan queue:work        (Queue processor)
Terminal 3:  npm run dev                   (Dev assets - optional)

Then open: http://localhost:8000
